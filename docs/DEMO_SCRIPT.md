> **평가 중점**: 데이터 아키텍처 / Mart 구성 / 데이터 제공 유연성 / AI
> **심사위원장**: 김영학 교수 (심장내과, 관상동맥질환 — 협심증/심근경색)
> **부위원장**: 양동현 교수 (영상의학과, CT/MRI 심장질환 진단)
> **평가자**: 위 2명 + IT 팀장급 3명
> **URL**: http://localhost:5173
> **화면전환**: 5회만 (대시보드→아키텍처탭→CDW→NER→데이터마트)

## 인트로 (5초)

**화면**: 대시보드 (미리 로그인)

> 아산병원 의료 데이터 통합 플랫폼입니다. **7만 6천 환자**, **9,300만 건** 임상 데이터가 적재되어 있습니다.

---

## 1. 데이터 아키텍처 (25초)

**화면**: [아키텍처 뷰] 탭 클릭

> 전체 데이터 흐름입니다. 위에 **정형**(EMR·처방·검사)과 **비정형**(PACS·병리) 원천 시스템이 있고, **5단계 파이프라인** — 수집, 정제, 표준 변환, 품질 검증, 연구 마트를 거칩니다. OMOP CDM 국제표준으로 자동 변환됩니다.
>
> 연구 마트에서 **통합 질의 엔진**을 통해 자연어 조회, BI 대시보드, AI 분석, 의무기록 분석 **4가지 서비스**로 제공됩니다.

---

## 2. 데이터 제공 유연성 (30초)

**화면**: `/cdw` 이동 → 미리 실행된 "급성 심근경색 환자의 연령별 분포" 결과

> 데이터 제공 방식입니다. 지금 보시는 것이 **자연어 질의**입니다. "급성 심근경색 환자 연령별 분포"라고 한국어로 질문하면, AI가 SQL을 자동 생성하고 실행합니다. IT 부서 거치지 않고 교수님이 직접 조회할 수 있습니다.
>
> 이 외에 **BI 대시보드**, **JupyterLab**(Python/R) 분석환경도 제공합니다. 기술 수준에 따라 선택할 수 있습니다.

---

## 3. AI (30초)

**화면**: `/ner` 이동 → 미리 실행된 심근경색 진료기록 분석 결과

> AI 분석입니다. 심장내과 경과기록을 입력하면, AI가 텍스트에서 **진단명·약물명·검사 항목**을 자동 추출합니다. 심근경색은 SNOMED CT, Aspirin은 RxNorm, Troponin은 LOINC — **국제 표준 코드**가 자동 부여됩니다.

**화면**: `/ontology` 이동 → 검색창에 "심근경색" 입력 → **자동으로 진단/질환 노드 선택 + 연결 그래프 확대**

> 추출된 코드가 온톨로지에서 어떤 관계를 갖는지 봅니다. "심근경색"을 검색하면 **자동으로** 진단 노드가 선택되고, 관상동맥 중재술, 항혈소판 약물, Troponin 검사가 연결된 **지식 그래프**가 확대됩니다.

---

## 4. Mart 구성 (25초)

**화면**: `/datamart` 이동 → 임상 데이터 현황 탭

> 실제 데이터마트입니다. **76,074명** 환자의 성별 분포, 입원·외래·응급 **450만 건** 방문 분포, 도메인별 데이터 품질 점수를 보실 수 있습니다. 아래에 **급성 심근경색 임상 경로**도 시각화되어 있습니다.
>
> 의료 영상 **28만 장**, 테이블별 스키마·샘플 데이터·Python/R 코드도 제공됩니다.
